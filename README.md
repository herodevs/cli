# HeroDevs CLI -- `@herodevs/cli`

## Installation

```
npm install -g @herodevs/cli
```

## Usage

after global installation

```
@herodevs/cli ____
```

or

```
hd ____
```

without installation:

```
npx @herodevs/cli ____
```

## Commands

Get a list of committers within a git repository

```
hd report committers
```
